Claude — AI by Anthropic

The best AI for vibecoding. Understands context deeply, writes clean code, explains everything.

Recommended ▼
1

Go to claude.ai

Open your browser and go to claude.ai. You can use Claude directly in the browser — no download needed for the web version.

Open claude.ai ↗
2

Create a free account

Sign up with your Google account or email. The free plan gives you plenty of messages to learn and build with.

3

Download Claude desktop app (optional)

For a better experience, download the Claude desktop app. Works on Windows and Mac. Faster, cleaner, and you can use it alongside your code editor.

Download Claude app ↗
4

Start vibecoding

Type exactly what you want to build. Be specific — the more detail you give, the better the code Claude writes for you.

Pro tip: Start your prompt with "Build me a..." and describe exactly what you want — the design, the features, the colors. Claude understands plain language perfectly.
⚡

Antigravity — AI Code Editor

A code editor built for vibecoding. Paste AI-generated code here, preview it live, edit and ship.

Recommended ▼
1

Download Antigravity

Go to the Antigravity website and download the app for your OS — Windows or Mac. It's free.

Download Antigravity ↗
2

Install and open it

Run the installer. Once open you'll see a clean editor. Create a new folder for your project from File → New Project.

3

Create your first file

Right click in the file panel → New File → name it index.html. This is where your code goes.

4

Paste code from Claude and preview

Copy code from Claude → paste in Antigravity → click the preview button to see it live in the browser instantly.

Pro tip: Use Claude to write the code and Antigravity to run and edit it. These two tools together are your complete vibecoding setup.
💜

Lovable — Just describe, it builds

Want to try vibecoding without any setup? Lovable builds full apps just from your description. No code editor needed.

Best for beginners ▼
1

Go to lovable.dev

No download needed. Lovable runs fully in the browser. Just open the website and sign up free.

Open Lovable ↗
2

Describe what you want to build

Type something like "Build me a personal portfolio website with dark theme and a projects section". Lovable builds it instantly — full app, live preview.

3

Edit by chatting

Don't like something? Just type "change the color to lime green" or "add a contact form". It updates live. No code knowledge needed at all.

4

Deploy with one click

Once happy with your project, click Deploy. Lovable hosts it for you and gives you a real URL to share. Free on the starter plan.

When to use Lovable: Perfect for quickly trying out an idea or building something fast without any setup. When you want more control and to actually learn — switch to Claude + Antigravity.
🖤

Blackbox AI — AI inside your editor

A VS Code extension that brings AI directly into your code editor. Autocomplete, explain code, fix bugs — all inside the editor.

Free ▼
1

Install VS Code first

Blackbox AI works as a VS Code extension. Download VS Code free if you don't have it.

Download VS Code ↗
2

Install Blackbox AI extension

Open VS Code → press Ctrl+Shift+X → search "Blackbox AI" → click Install. Done in 30 seconds.

Visit Blackbox AI ↗
3

Start coding with AI help

As you type code, Blackbox suggests completions automatically. You can also select any code and ask it to explain or fix it.

When to use Blackbox: Great when you're writing code yourself and want AI suggestions inline. Works best alongside Claude for the full vibecoding experience.
